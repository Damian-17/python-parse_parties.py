# Парсер политических партий РФ

Парсер для извлечения данных о зарегистрированных политических партиях Российской Федерации с официального сайта Министерства юстиции.

## Функциональность

- Парсинг HTML-страницы со списком политических партий
- Извлечение наименований партий и ссылок на их документы
- Автоматическое преобразование относительных ссылок в абсолютные
- Обработка и очистка URL (исправление протокола, удаление параметров)
- Сохранение данных в структурированном виде (Python dict/JSON)

##  Извлекаемые данные

Для каждой политической партии извлекается:
- **Наименование** (полное название партии)
- **Ссылка на документ** (устав, регистрационные сведения) или `None`, если документ отсутствует

##  Требования

- Python 3.6 или выше
- Установленные зависимости из `requirements.txt`
